# BWCryptoFramework
A Cryptography Framework for Tibco Businessworks

## Custom Functions:
  - signSHA256withRSA
  - verifySHA256withRSA
  - encryptRSACipher
  - decryptRSACipher
    
## Java Methods:
  - encryptAES256ForFile
  - decryptAES256ForFile
  - generateECDHKeys
  - generateECDHSecret
  - generateSHA512ForFile
  - verifySHA512ForFile
  - zipFiles
  - unzipFiles
